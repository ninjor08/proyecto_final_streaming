# Proyecto Final Streaming (CineStream)

Aplicación móvil desarrollada en Flutter y Firebase que permite a los usuarios autenticarse mediante Firebase Authentication, visualizar un catálogo de películas almacenado en Firebase Realtime Database y reproducir contenido multimedia mediante Video Player.

## Tecnologías utilizadas

- Flutter
- Firebase Authentication
- Firebase Realtime Database
- Video Player
- Material Design

## Funcionalidades

- Inicio de sesión
- Catálogo de películas
- Reproducción de video
- Integración con Firebase
